##NetTXTEZ是什么?
一个针对小说网站开发的爬虫下载器

##NetTXTEZ有哪些功能？

- [x] 方便的下载功能
- [x] 便捷的Text文档保存功能
- [x] 超多支持的小说（只有笔趣阁，有时间开发更多23333
- [x] 支持下载中断和续写功能
